My Projects

Here a few screenshots of some projects I have done so far in my program! If you want to check out more stuff I have done, check out my GitHub!

Project 1: Turn Based Battle Simulator

This is a simple command line program that that takes random names, titles, and weapons and creates two "warriors' to fight randomly in a best two out of three fight.

Project 2: Student Management System

Another simple command line program. This one takes input from the user and allows them to add students, read details about students, update student details, and remove students.

Project 3: Student Grades Management System

This simple command line program, allows users to enter 4 grades of a student and find their average. It also takes the average of all students entered as well upon ending program.

Project 4: Python Game

This is a simple Python game, that I created using a book as a guide. It is based off of the game "Space Invaders". It uses OOP concepts such as classes, inheritance, and polymorphism to create the game mechanics and enemy behavior.

Project 5: C++ Project

This is a simple command line program written in C++. It is a simple game where you guess if the card is higher or lower. It uses classes, vectors, arrays, and basic control logic to randomly draw cards from a deck and the player decideds if the next card is lower or higher.
